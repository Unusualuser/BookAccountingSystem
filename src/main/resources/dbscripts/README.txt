Для успешного выполнения bat-скрипта, необходимо:
    - положить папку scripts и файл run.bat в корневую директорию PostgreSQL (в папку bin)
    - изменить в bat-скрипте, имя пользователя у флага -U с 'postgres' на то, которое у вас, а так же в файле createDatabase у атрибута "OWNER"
    - запустить bat-скрипт из консоли и ввести пароль от пользователя postgres`а необходимое число раз
